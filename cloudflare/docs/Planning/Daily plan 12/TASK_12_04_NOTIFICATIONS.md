# TASK 12-04: Notification System (Toasts)

## Objective
Implement a toast notification system to provide feedback for user actions and API responses.

## Requirements
- Support different notification types (Success, Error, Info, Warning).
- Implement auto-dismissal and manual close.
- Provide clear and actionable messages.
- Integrate with the existing error handling logic.

## Deliverables
- Reusable toast component.
- Integrated notification service in the frontend.
