# TASK 12-03: Refinement & Versioning (UI)

## Objective
Implement the user interface for conversational refinement and version switching.

## Requirements
- **Version Selection UI**: A component to allow users to switch between and preview different generation versions.
- **Refinement Context UI**: A UI that shows the context being used for the current refinement turn.
- **Iteration Feedback**: Visual cues that indicate a refinement is in progress.

## Deliverables
- UI components in the `portal` for version management and refinement context.
