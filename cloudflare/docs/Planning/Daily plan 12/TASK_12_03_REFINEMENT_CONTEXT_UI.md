# TASK 12-03: Refinement Context UI

## Objective
Implement a UI component that shows the context (e.g., user instructions) for the current refinement turn.

## Requirements
- Display the most recent user prompt/instruction.
- Allow users to edit the instruction for the next turn.
- Provide visual feedback when the agent is "thinking" about the refinement.

## Deliverables
- UI component for refinement context in the Builder.
